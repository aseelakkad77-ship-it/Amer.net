# Amer.net - تطبيق تسجيل الديون

تطبيق احترافي لتسجيل ديون الزبائن داخل المحل، يعتمد بالكامل على التخزين المحلي دون أي اتصال بالإنترنت.

## المميزات

- ✅ تطبيق 100% محلي بدون إنترنت
- ✅ قاعدة بيانات سريعة (Isar)
- ✅ واجهة عصرية واحترافية باللغة العربية
- ✅ دعم الوضع الليلي
- ✅ بحث فوري
- ✅ سجل كامل للعمليات
- ✅ إحصائيات شاملة
- ✅ نسخ احتياطي واستيراد

## المتطلبات

- Flutter 3.0+
- Dart 3.0+

## التثبيت

```bash
flutter pub get
flutter run
```

## البنية

```
lib/
├── config/          # إعدادات التطبيق والمواضيع
├── models/          # نماذج البيانات
├── providers/       # State Management
├── screens/         # الشاشات الرئيسية
├── services/        # خدمات قاعدة البيانات
├── widgets/         # المكونات المشتركة
└── main.dart        # نقطة البداية
```

## الشاشات

1. **الشاشة الرئيسية** - عرض جميع العملاء مع البحث السريع
2. **إضافة عميل** - إضافة أو تعديل بيانات العميل
3. **تفاصيل العميل** - عرض الرصيد والعمليات
4. **إضافة عملية** - تسجيل دين أو تسديد
5. **الإحصائيات** - عرض الإحصائيات الشاملة
6. **النسخ الاحتياطي** - تصدير واستيراد البيانات
