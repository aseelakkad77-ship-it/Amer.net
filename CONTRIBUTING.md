# Amer.net - دليل المساهمة

شكراً على اهتمامك بالمساهمة في مشروع Amer.net! 🎉

## خطوات المساهمة

### 1. Fork المستودع
اضغط على زر Fork في أعلى الصفحة

### 2. استنساخ المستودع الخاص بك
```bash
git clone https://github.com/YOUR_USERNAME/Amer.net.git
cd Amer.net
```

### 3. إنشاء فرع للميزة الجديدة
```bash
git checkout -b feature/your-feature-name
```

### 4. إجراء التغييرات
- اتبع معايير الكود الموجودة
- تأكد من أن الكود يعمل بشكل صحيح
- أضف تعليقات واضحة

### 5. الالتزام بالتغييرات
```bash
git add .
git commit -m "Add: وصف واضح للتغيير"
```

### 6. دفع التغييرات
```bash
git push origin feature/your-feature-name
```

### 7. إنشاء Pull Request
اذهب إلى المستودع الأصلي وإنشاء PR

## معايير الكود

- استخدم Dart format
- اتبع Flutter/Dart conventions
- اكتب تعليقات واضحة
- اختبر الكود قبل الالتزام

## الإبلاغ عن المشاكل

إذا وجدت مشكلة:
1. اذهب إلى GitHub Issues
2. تحقق من عدم وجود issue مماثل
3. أنشئ issue جديد مع وصف واضح

شكراً لمساهمتك! ❤️
